ShopNScroll

*An E-Commerce Site that focuses on multiplatform easy-to-browse and easy-to-buy UI to widen the age-group of Customers.

#Tools Used: 
  *Eclipse IDE (Neon.2)
  *H2 Console
  
#Technologies Used:
  *Maven
  *Spring MVC 4 
  *Hibernate 5 
  *Java Server Pages

#Languages and Framework:
-Front End:
  *HTML 5
  *CSS 3
  *Javascript 1.8
  *Bootstrap 4
  *JQuery 3

-Back End: 
  *Java
  *HQL

#Databases Used: *H2 DB
